## Installation and Build

`npm install && ./node_modules/grunt/bin/grunt release`

## Start the server

`node server.js`

## Visit the Apache Nifi Registry Explorer UI

`http://localhost:3000/nifi-registry/explorer/grid-list/23f6cc59-0156-1000-06b4-2b0810089090`

## Visit the Nifi Registry Fluid Design System.

`http://localhost:3000/nifi-registry/fluid-design-system`
